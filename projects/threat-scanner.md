# Threat Scanner

<img src="{{ '/assets/img/projects/threat-scanner.svg' | relative_url }}" alt="Threat Scanner preview" style="max-width:100%;border-radius:8px;margin-bottom:12px">

An automated scanner that detects suspicious configurations and common misconfigurations across cloud and on-prem systems.

Features:
- Multi-engine scanning (Nmap, custom Python checks)
- Configuration checks (STIG/SCAP patterns)
- Exportable reports and remediation guidance

Technologies: Python, Nmap, Flask (UI)

Repository: https://github.com/nkojoedou-del/threat-scanner
